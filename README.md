# 🛡️ Especialización en Ciberseguridad | UCAM & ThePower
**Itinerario de Especialización Técnica en Seguridad, Programación y Datos**

Este repositorio centraliza los proyectos y laboratorios desarrollados durante el programa de especialización avalado por la **UCAM (Universidad Católica de Murcia)**. El enfoque de este itinerario es dotar al Administrador de Sistemas de las capacidades de programación y gestión de datos necesarias para afrontar los retos de la ciberseguridad moderna.

---

## 🎯 Visión Estratégica
Como alumno de **ASIR**, entiendo que la ciberseguridad no es una capa aislada, sino una competencia transversal. Este programa me permite bajar al nivel del código para:
* **Entender la superficie de ataque:** Analizar vulnerabilidades en aplicaciones web (HTML/CSS/JS).
* **Automatizar la defensa:** Crear scripts en **Python** para la respuesta ante incidentes y análisis de logs.
* **Securizar el dato:** Comprender las inyecciones SQL y la arquitectura de bases de datos NoSQL para proteger la persistencia de la información.

---

## 🛠️ Proyectos y Módulos de Especialización

### 1. 📂 [Portfolio Profesional (HTML5 & CSS3)](./1_Portfolio_HTML_CSS)
Desarrollo de una interfaz web profesional para la exposición de activos y proyectos.
* **Foco:** Semántica, accesibilidad y diseño responsive.
* **Cyber-link:** Entender la estructura del DOM es el primer paso para detectar ataques XSS (Cross-Site Scripting) y defacements.

### 2. 📂 [Lógica de Programación: Katas JavaScript](./2_Katas_JS)
Resolución de retos técnicos centrados en la lógica de programación asíncrona y manipulación del DOM.
* **Foco:** ES6+, asincronía y validación de datos en cliente.
* **Cyber-link:** JavaScript es el lenguaje del navegador; dominarlo permite auditar scripts maliciosos y entender el comportamiento de payloads en cliente.

### 3. 📂 [Automatización y Scripting: Katas Python](./3_Katas_Python)
Desarrollo de algoritmos, manejo de estructuras de datos y Programación Orientada a Objetos (POO).
* **Foco:** Funciones lambda, filter/map/reduce, manejo de excepciones y módulos.
* **Cyber-link:** Python es la navaja suiza de la ciberseguridad. Estos ejercicios sientan las bases para desarrollar herramientas propias de escaneo, parsing de logs y explotación de vulnerabilidades.

### 4. 📂 [Gestión de Datos: SQL & NoSQL](./4_Bases_Datos)
Administración de sistemas de persistencia relacionales y no relacionales.
* **SQL (PostgreSQL):** Queries complejas, gestión de reservas y auditoría de integridad.
* **NoSQL (MongoDB/Node.js):** Desarrollo de un CRUD completo para gestión de servicios, control de errores y APIs REST.
* **Cyber-link:** Protección contra SQL Injection y comprensión de la seguridad en bases de datos orientadas a documentos.
